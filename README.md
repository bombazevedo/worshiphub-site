## Placeholders para substituir
No `index.html` e `script.js`, substitua:
- `__PLAY_STORE_URL__`
- `__WHATSAPP_LINK_GENERIC__`
- `__PRIVACY_URL__`
- `__TERMS_URL__`

### WhatsApp
Use a base neste formato:
`https://wa.me/55SEUNUMERO?text=`

O `script.js` acrescenta automaticamente a mensagem do plano.
